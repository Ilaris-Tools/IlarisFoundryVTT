# Geführte Touren im Ilaris-System

Das Ilaris-System für Foundry VTT bietet geführte Touren an, um neue Benutzer durch die wichtigsten Funktionen zu führen.

## Verfügbare Touren

### Spezialdialoge im Ilaris-System

Diese Tour führt Sie durch die wichtigsten Spezialdialoge im Ilaris-System.

**Was behandelt wird:**

-   Fertigkeitsprobe-Dialog
-   Nahkampf-Dialog
-   Fernkampf-Dialog
-   Dialog für Übernatürliche Fertigkeiten (Zauber, Liturgien, etc.)
-   Manöver-System
-   Ziel-Auswahl und Automatisierung
-   Live-Vorschau-Funktion
-   Tipps und Tricks

**Wo zu finden:**

1. Öffnen Sie in Foundry VTT das Kompendium-Menü (rechte Sidebar)
2. Suchen Sie nach dem Kompendium "Touren"
3. Öffnen Sie die Tour "Spezialdialoge im Ilaris-System"
4. Klicken Sie auf "Tour starten"

**Für wen geeignet:**

-   Neue Spieler, die das Ilaris-System kennenlernen möchten
-   Spielleiter, die ihren Spielern das System erklären möchten
-   Erfahrene Benutzer, die Details zu speziellen Dialogen nachlesen möchten

## Dialoge im Detail

### Fertigkeitsprobe-Dialog

**Aufruf:** Charakterblatt → Fertigkeiten-Tab → Auf eine Fertigkeit klicken

**Funktionen:**

-   Würfeln von Fertigkeitsproben und Attributsproben
-   Anwendung von Modifikatoren (Boni/Mali)
-   Einsatz von Schicksalspunkten (3W20)
-   Auswahl von Talenten

### Nahkampf-Dialog

**Aufruf:** Charakterblatt → Kampf-Tab → Auf eine ausgerüstete Nahkampfwaffe klicken

**Funktionen:**

-   Angriffswurf (AT)
-   Verteidigungswurf (VT)
-   Schadensberechnung
-   Auswahl von Kampfmanövern
-   Berücksichtigung von Lichtverhältnissen
-   Zielauswahl

**Wichtig:**

-   Waffen müssen ausgerüstet sein (Symbol links neben dem Namen anklicken)
-   Zweihändige Waffen müssen sowohl als Haupt- als auch als Nebenwaffe ausgerüstet sein

### Fernkampf-Dialog

**Aufruf:** Charakterblatt → Kampf-Tab → Auf eine ausgerüstete Fernkampfwaffe klicken

**Funktionen:**

-   Fernkampfangriff (FK)
-   Reichweiten-Modifikatoren (Kurz/Mittel/Weit)
-   Größenklasse des Ziels
-   Lichtverhältnisse, Wetter, Bewegung
-   Deckung und Kampfgetümmel
-   Automatische Entfernungsberechnung (wenn Ziele ausgewählt)

### Dialog für Übernatürliche Fertigkeiten

**Aufruf:** Charakterblatt → Übernatürliches-Tab → Auf einen Zauber/Liturgie klicken

**Funktionen:**

-   Probenwurf für übernatürliche Fertigkeiten
-   Verwaltung von AsP/KaP (Astral-/Karmapunkte)
-   Energiekosten-Berechnung
-   Spezielle Manöver (Blutmagie, Verbotene Pforten, etc.)
-   Angriffswürfe für offensive Zauber

**Wichtig:**

-   AsP/KaP werden nur angezeigt, wenn der Vorteil "Zauberer" oder "Geweiht" aktiviert ist
-   Zukauf von AsP/KaP kann durch Mausklick auf das Label eingestellt werden

## Manöver

Manöver sind spezielle Kampftechniken oder Zaubermodifikationen, die zusätzliche taktische Optionen bieten.

**Freischaltung:**

-   Manöver werden NICHT durch die Manöver im Kompendium freigeschaltet
-   Freischaltung erfolgt über die entsprechenden Vorteile/Sonderfertigkeiten

**Voraussetzungen:**

-   Waffeneigenschaften
-   Ausgerüstete Waffe
-   Bestimmter Kampfstil
-   Entsprechende Sonderfertigkeit

**Beispiele:**

-   Nahkampf: Finte, Sturmangriff, Niederwerfen
-   Fernkampf: Gezielter Schuss, Schnellschuss
-   Magie: Blutmagie, Verbotene Pforten

## Live-Vorschau

Alle modernen Dialoge zeigen eine Live-Vorschau der Würfelformel und der finalen Werte.

**Was wird angezeigt:**

-   Vollständige Würfelformel (z.B. "1d20 + 12")
-   Aufschlüsselung aller Modifikatoren
-   Basiswerte (PW, AT, VT, FK)
-   Angewendete Boni und Mali
-   Energiekosten (bei übernatürlichen Fertigkeiten)

**Klickbare Zusammenfassungen:**
In vielen Dialogen können Sie direkt auf die Zusammenfassung klicken, um den entsprechenden Wurf auszuführen.

## Ziel-Auswahl

Das System kann Ziele automatisch aus der Foundry-Auswahl übernehmen.

**Aktivierung:**
Systemeinstellungen → Automatisierung → Zielauswahl aktivieren

**Verwendung:**

1. Wählen Sie auf der Szene ein oder mehrere Token als Ziele aus (Rechtsklick → "Ziel")
2. Öffnen Sie einen Dialog
3. Die Ziele werden automatisch angezeigt
4. Die Entfernung wird automatisch berechnet

## Häufige Probleme

**Waffen nicht auswählbar:**

-   Waffe muss ausgerüstet sein (Symbol links neben dem Namen anklicken)

**Rüstung wird nicht berechnet:**

-   Rüstung muss angelegt werden

**Manöver fehlen:**

-   Prüfen Sie Voraussetzungen und Vorteile

**Keine Schicksalspunkte-Option:**

-   Charakter muss Schicksalspunkte haben

**AsP/KaP nicht sichtbar:**

-   Vorteil "Zauberer" oder "Geweiht" muss aktiviert sein
